# Mathematics for Machine Learning (Imperial College London)

This repository contains my personal Python implementations, programming labs, and mathematical applications completed during the Imperial College London curriculum on Coursera. This coursework serves as the quantitative foundation for my financial engineering and machine learning models.

## 🛠️ Core Skills Mastered
* **Linear Algebra:** Vector spaces, matrix transformations, orthogonal projections, basis changes, and Eigenvalues/Eigenvectors ($Ax = \lambda x$).
* **Multivariate Calculus:** Partial derivatives, gradients, Jacobians, Hessians, and their application in optimization algorithms (Gradient Descent).
* **Principal Component Analysis (PCA):** Mathematical dimensionality reduction, data variance retention, and noise filtering for large datasets.
